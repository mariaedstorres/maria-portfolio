# Maria Torres — Personal Portfolio ✨

A clean, single-page portfolio website built to share my journey from **Industrial Mechanics** into **Tech**. It introduces who I am, what I'm learning, and how to get in touch — all wrapped in a soft, minimal design with light/dark mode.

## 🪞 What it is

This is my personal portfolio — a digital space documenting my transition into software development. It's built from scratch with plain HTML, CSS, and JavaScript (no frameworks), as both a learning exercise and a real-world calling card.

## 🧩 Sections

- **Hero** — quick intro and call-to-action buttons
- **About Me** — my story and career transition
- **Skills** — languages, tools, and what I'm currently learning
- **Projects** — a growing showcase of my work
- **Beyond Code** — a more personal side (my dog Bela, books, gym, music)
- **Quotes** — favorite quotes that keep me motivated
- **Contact** — GitHub, LinkedIn, and email

## 🛠️ Built with

- **HTML5** for structure
- **CSS3** for styling, animations, and the light/dark theme
- **JavaScript** for interactivity (theme toggle, mobile menu, scroll reveals)
- [Lucide Icons](https://lucide.dev/) for icons
- Google Fonts (Cormorant Garamond, DM Sans, Playfair Display)

## 🚀 Running locally

No build step required — just open the file in a browser:

```bash
git clone https://github.com/mariaedstorres/maria-portfolio.git
cd maria-portfolio
```

Then open `index.html` in your browser, or serve it locally:

```bash
npx serve .
```
